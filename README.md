# Web Calculator

A web application for performing basic arithmetic operations directly in your browser. No installation required.

## Features

*   **Basic Arithmetic:** Perform addition, subtraction, multiplication, and division with ease.
*   **Responsive Design:** Fully functional on both desktop and mobile devices.
*   **Clean UI:** Minimalist interface designed for focus and usability.

## Project Structure

*   `calculator.html` — The core structure and layout of the application.
*   `style.css` — Visual styling, including the grid layout and button effects.
*   `script.js` — The "brains" of the operation: handles calculation logic and user input.

## Requirements

*   Any modern web browser (Chrome, Firefox, Safari, or Edge).

## How to Run

### Live Demo
You can access the calculator immediately here: 
👉 [**https://froymi.github.io/calculator/calculator.html**](https://froymi.github.io/calculator/calculator.html)

### Local Setup
1.  **Clone the repository:**
    ```bash
    git clone https://github.com/froymi/calculator.git
    ```
2.  **Open the project:**
    Navigate to the project folder and open `calculator.html` in your preferred browser.
